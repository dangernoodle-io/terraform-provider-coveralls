# terraform-provider-coveralls

Terraform provider for [Coveralls.io](https://coveralls.io), managing repository configurations.

Registry: `registry.terraform.io/dangernoodle-io/coveralls`

## Stack

- Go 1.25, terraform-plugin-framework v1 (protocol v6), no CGO
- HTTP client: `go-resty/resty`
- Test mocking: `jarcoal/httpmock`

## Structure

```
internal/provider/
  client/          # Coveralls API wrapper (Create/Get/Update)
  provider.go      # Provider registration, token config
  resource_repository.go
  data_source_repository.go
tools/             # tfplugindocs codegen (build:generate)
examples/          # Terraform usage examples
docs/              # Auto-generated — do not hand-edit
```

## Commands

| Command | Description |
|---|---|
| `make` | fmt + lint + install + generate |
| `make test` | Unit tests (httpmock, no network) |
| `make testacc` | Acceptance tests (real API, requires env vars below) |

Acceptance test env vars: `TF_ACC=1`, `COVERALLS_API_TOKEN`

## Resources & Data Sources

### `coveralls_repository`

| Attribute | Required | Notes |
|---|---|---|
| `name` | yes | `owner/repo` format |
| `service` | yes | e.g. `github` |
| `comment_on_pull_requests` | yes | |
| `send_build_status` | yes | |
| `commit_status_fail_threshold` | no | Float |
| `commit_status_fail_change_threshold` | no | Float |
| `token`, `created_at`, `updated_at`, `id` | computed | |

Import: `terraform import coveralls_repository.example github:owner/repo`

**Delete is not supported by the Coveralls API** — destroy only removes from state.

### `coveralls_repository` (data source)

Required: `name`, `service`. All other fields computed.
